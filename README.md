# 📽️ دانلودر یوتیوب
با استفاده از این پروژه شما میتوانید ویدیو های یوتوب را در فضای شخصی خود در گیتهاب نگهدارید و سپس در مواقع نیاز آنرا دانلود کنید. 
## قابلیت های این پروژه
- 💾 ذخیره سازی اتوماتیک فایل ها در فضای شخصی شما
- 🏙️ نگهداری دائمی در فضای شخصی گیتهاب شما
- 📂 پارت بندی کردن ویدیو های طولانی به صورت zip
- ⌨️ دانلود خودکار زیرنویس و ذخیره در subtitle.zip
- 🗃️ دسته بندی ویدیوها در پوشه های مختلف
- 📼 امکان دانلود صوت ویدیو به صورت جداگانه
- 🔗 پشتیبانی از دانلود همزمان چند url (جداسازی با فاصله)
- 🔒 امکان رمزگذاری روی فایل ها جهت محافظت
- 🖼️ قابلیت تنظیم کیفیت ویدیو دانلودی

> [!bug] باگ های موجود در این نسخه
> 1. در بعضی موارد ویدیو های دانلود شده با کیفیت بالا مشکل Playback دارند.
> 2. در بعضی موارد پاکسازی تاریخچه به درستی صورت نمیگیرد.

## ⚓ آموزش نصب 
- ابتدا وارد اکانت گیتهاب خود شوید.
- پروژه را fork کنید. (در بالای صفحه سمت راست)
- نام fork خود را انتخاب کنید و  روی create fork کلیک کنید.
## 🪴 آموزش استفاده 
### وارد نسخه fork خود شوید
معمولا این کار به صورت خودکار صورت میگیرد.
شما میتوانید پروژه fork شده خود را در فضای شخصی خود در بخش Repositories پیدا کنید.
### وارد تب Actions شوید
> استفاده از قابلیت های این پروژه در تب Actions امکان پذیر است!
#### دانلود ویدیو یوتوب!
> با استفاده از این قابلیت شما میتوانید ویدیو های یوتوب خود را دانلود کنید!
- روی این قابلیت کلیک کنید!
- در صفحه‌ی باز شده روی run workflow کلیک کنید.
- آدرس ویدیو یوتیوب یا آدرس های یوتیوب(جداسازی با فاصله) را در باکس "لینک ویدیو" وارد کنید.
- اگر میخواهید زیرنویس همراه ویدیو دانلود شود تیک آن را بزنید.
- کیفیت ویدیو را از باکس "کیفیت" انتخاب کنید.
- درصورت نیاز برای فایل خود رمز تعیین کنید.
- روی دکمه ی run workflow در زیر باکس ها کلیک کنید.
- منتظر اتمام کار و مشاهده تیک سبز بمانید.

> [!info]- نکات
> 1. زیر نویس های ویدیو فقط به زبان های فارسی و انگلیسی و درصورت وجود داشتن دانلود میشود!
> 2. برای دانلود صوت ویدیو به صورت جداگانه از باکس کیفیت audio را انتخاب کنید!
> 3. در صورت خالی گداشتن باکس رمز، رمزی برای فایل شما گداشته نمیشود.
> 4. بین 2 تا 5 دقیقه طول میکشد تا تمام مراحل workflow تمام شود.

> [!important]- کیفیت ویدیو
>
> کیفیت انتخاب شده توسط شما، ماکسیمم کیفیتی است که شما از ویدیو میخواهید!
> این به این معناست که اگر کیفیت درخواستی شما برای ویدیو موجود نباشد، بهترین کیفیت پایین تر از آن دانلود میشود.
> شما میتوانید با انتخاب گزینه best بهترین کیفیت موجود برای ویدیو را دانلود کنید!

> [!warning] لینک ها
>
> حتما از لینک با دامنه خود سایت یوتوب برای دانلود ویدیو ها استفاده کنید.
> شما میتوانید از گوگل ویدیو هارا پیداکنید و لینک آن را کپی کنید.

#### محل ذخیره سازی ویدیو ها!
در نسخه fork شده خود وارد تب Code شوید و وارد پوشه videos شوید.
درصورتی که ویدیو دانلود کرده باشید لینک پیمایش به صفحه فایل های ویدیو در لیست مجود میباشد.
شما میتوانید به صورت دستی به محل ذخیره ویدیو خود پیمایش کنید.
در فایل README موجود در فولدر مخصوص به ویدیو شما لینک دانلود تمام پارت ها قرار گرفته.

> [!tldr]- آموزش ذخیره ویدیوها
> 1. ابتدا تمامی پارت ها را دانلود کنید.
> 2. سپس تمام پارت های دانلود شده را در یک پوشه قرار دهید.
> 3. توسط نرم افزار دلخواه خود (ترجیحا WinRAR) فقط فایل با پسوند zip را استخراج کنید.
> 4. ویدیو شما در پوشه قرار دارد.

> [!info] ذخیره فایل ها در شرایط فعلی اینترنت کشور
>
> در بعضی تست ها امکان دانلود فایل های موجود در پوشه videos امکان پذیر نبود.
> چه از طریق لینک موجود در README و چه از طریق دانلود Raw
> اما به وسیله Clone کردن و Fetch کردن فایل ها توسط نرم افزار GitHub دسکتاپ میتوانید فایل ها را در سیستم خود ذخیره کنید.
> - ابتدا نرم افزار GitHub Desktop را در دستگاه دستکتاپ خود نصب کنید.
> - سپس وارد اکانت خود شوید.
> - ریپازیتوری fork شده MyTube را انتخاب کرده و آن را در مسیر مشخص در دستگاه خود Clone کنید.
> - حال تمامی فایل های موجود در فضای شخصی نسخه fork شده شما در مسیر استخاب شده در سیستم شما موجود میباشد.
> - از این به بعد بعد از دانلود های جدید میتوانید با Fetch کردن و Pull کردن، فایل های جدید را در سیستم خود ذخیره کنید!

# ⬇️ دانلود فایل در شرایط اینترنت بین المللی

اگر توانایی استفاده از این پروژه را ندارید و یا فایل خاصی بصورت ضروری مورد نیاز شما میباشد، میتوانید نام آن فایل یا لینک دانلود آنرا به پشتیبانی ما ارسال کنید تا در صورت امکان برای شما تهیه شده و با اینترنت داخلی قابل دانلود باشد. لینک یا نام فایل خود را از طریق پیام رسان بله به آیدی `Mquency_Support@` ارسال کنید.
